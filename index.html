Step 1: Install Forge

We are running on Minecraft 1.20.1.

  1. Download the Forge Installer (1.20.1).
  2. Run the downloaded .jar file.
  3. Select "Install client" and click OK.
Step 2: Download & Sync

We use a custom sync script to manage mods automatically.

  1. Download the Pack: Click the "Code" -> "Download ZIP" button on the GitHub repository.
  2. Extract: Unzip the folder to your Documents or Games folder.
  3. Run: Double-click update-mods-client.bat.

This script will download all mods directly to your Minecraft folder.

  4. Find JVM Arguments. Change -Xmx2G to -Xmx8G.
  5. Click Save.
Step 4: Play
  • Select your profile and hit Play.
  • Go to Multiplayer -> Add Server.
